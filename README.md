# Weather Reports
gRPC stack for Raspberry Pi environment sensors in my yard

## Devices
### Setup
```bash
git clone https://github.com/jpobley/weather_reports.git
cd weather_reports
sudo -s
source build.sh
exit # get out of sudo mode
```

### Update
```bash
cd /path/to/weather_reports/repo
git pull
sudo -s
source build.sh
exit # get out of sudo mode
```
